This repository contains deployment guides and configuration files for setting up Open Air Interface (OAI) 5G components integrated with IOS-MCN management and orchestration stack.

## Quick Navigation

1. Start with [oai-cn5g.md](oai-cn5g.md) to deploy the OAI 5G Core Network using Docker containers.
2. Follow [oai-ran.md](oai-ran.md) to deploy gNB and NR UE in RF simulator mode.
3. Use [iosmcn-o1-adapter.md](iosmcn-o1-adapter.md) to deploy the IOS-MCN O1 Adapter.
4. Deploy [iosmcn-smo.md](iosmcn-smo.md) IOS-MCN SMO stack.

For detailed system requirements, refer to individual deployment guides.
